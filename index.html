Hvordan fungerer den gratis prøveperiode?

Du kan prøve NemComply gratis uden at indtaste kreditkortoplysninger. Du betaler kun for de klienter du screener — 20 kr. pr. klient pr. år. Ingen binding, ingen abonnement.

Hvilke PEP- og sanktionslister screener I mod?

Vi screener mod 17+ kilder inkl. danske PEP-lister (Folketinget, kommunalpolitikere), britiske PEP-lister, EU- og FN-sanktionslister, OpenSanctions, Wikidata, FINMA og flere. Listerne opdateres løbende.

Er mine data sikre hos NemComply?

Ja. Alle data krypteres og vi er fuldt GDPR-compliant. Data opbevares i Danmark på sikre servere. Vi har en komplet audit trail, så du altid kan dokumentere din databehandling.

Kan jeg integrere NemComply med mine eksisterende systemer?

Ja. NemComply har en dokumenteret REST API med token-baseret autentificering, så du kan integrere screening og klientoprettelse direkte i dine egne systemer. Du kan også importere klienter via CSV.

Hvem skal bruge KYC-screening?

Alle virksomheder omfattet af hvidvaskloven skal foretage KYC-screening. Det inkluderer bl.a. fondsmæglere, investeringsrådgivere, revisorer, advokater, ejendomsmæglere og andre finansielle virksomheder.
